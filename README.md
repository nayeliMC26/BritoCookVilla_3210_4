## Instructions
Change directory to this project folder, then run the following commands:\
`npm install --save three`

`npm install --save-dev vite`

Finally, run

`npx vite`

## Controls

'W': Walk forward

'A': Walk left

'S': Walk backward

'D': Walk right

'F': Toggle flashlight

'ESC': Toggle pause menu
